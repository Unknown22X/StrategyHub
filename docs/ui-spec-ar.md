# مواصفة واجهة التحكم العربية

## الهدف

واجهة سطح مكتب عربية أولاً لإدارة RangeBot، وليست واجهة تشخيص أو لوحة JSON.
تظل كل قرارات التداول والحماية في المحرك المحلي؛ تعرض الواجهة الحالة وتطلب
العمليات المؤكدة فقط.

## قواعد التصميم

- اتجاه النافذة والتخطيط من اليمين إلى اليسار، مع عزل الرموز والأسعار والنسب
  ووقت UTC داخل مقاطع LTR.
- خط التطبيق مركزي عبر `RangeBotWindow.set_arabic_font_family` لكي يمكن إدخال
  خط عربي مخصص لاحقاً من دون تعديل الصفحات.
- ثلاث حالات واضحة دائماً: الاتصال بالمحرك، نمط التشغيل، وحالة الأمان. لا تخفي
  رسالة المنع سببها أو العملية التي تبقى مسموحة.
- الأحمر للطوارئ والخطر، الكهرماني للتحذير أو الاستعداد الناقص، والأخضر للحالة
  الجاهزة؛ لا يكون اللون هو وسيلة التواصل الوحيدة.
- لا تظهر حقول JSON أو استجابات خام للمستخدم. تعرض الأخطاء بصياغة عربية قصيرة
  وتفاصيل القرار في جداول وبطاقات قابلة للقراءة.

## خريطة الصفحات

1. **الرئيسية**: بطاقات المحرك، الاتصال، النمط، Live Locked، العقد النشط،
   الرصيد، المركز، الحماية، التهدئة والمخاطر اليومية، مع شريط طوارئ ثابت.
2. **المراقبة**: بحث وإضافة العقود، جدول العقود المراقبة، الحالة والحداثة
   وقرار Long/Short، وتعيين العقد النشط والتشغيل التلقائي.
3. **تفاصيل القرار**: جدول فتح/آخر/أعلى/أدنى والنطاق والقرب، وقائمة شروط
   النجاح والمنع وسبب عربي.
4. **دخول يدوي**: نموذج الاتجاه والنوع والسعر، معاينة مالية، ثم حوار تأكيد.
5. **المركز والحماية**: المركز القائم وTP/SL، إغلاق وإلغاء، وحوارات منفصلة
   للتعديلات عالية الخطورة.
6. **المخاطر والطوارئ**: حدود اليوم والتهدئة، Emergency Stop وRESUME،
   وإغلاق طارئ مؤكد.
7. **السجل والمساعدة**: نشاط عربي، ملفات الإعداد، حالة تحقق Paper/Testnet،
   ومساعدة موضوعية قابلة للتصفح.

## تأكيدات السلامة

كل عملية تغيّر حالة تفتح حواراً يذكر العقد والنمط والنتيجة. تظهر عمليات Live
الحساسة حقل النص المطلوب حرفياً (`LIVE` أو `UNPROTECTED POSITION` أو
`DISABLE TP` أو `DISABLE SL`). لا تستطيع الواجهة تجاوز تحقق المحرك.

## أسطح الاختبار المتفق عليها

- عقد HTTP المحلي للعرض والعمليات المؤكدة.
- عناصر Qt العامة: تبويبات التنقل، بطاقات الحالة، حقول النماذج وحوارات التأكيد.
- لا تختبر الاختبارات تفاصيل التخطيط الداخلية أو آليات التخزين.
