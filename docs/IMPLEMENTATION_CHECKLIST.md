## Global rules

- Preserve valid Range Strategy behavior, risk controls, position protection, reconciliation, and user data.
- Use incremental refactoring; do not replace stable components only to change technology.
- Gate.io remains the trading platform and source of truth for live trading/account data.
- The frontend remains a localhost-only control panel; critical trading logic stays in the Python engine.
- Automated tests use mocks, Paper, or Testnet only. Never use real Live credentials in tests.
- Do not mark an item VERIFIED unless its stated verification was actually completed.
- Keep this file updated after each implementation slice.
